Step 1 source code

Assuming that you read everything in the previous sections, lets get started: navigate to this element This is a basic element which you can check out to see a basic polymer element in action

You can check the source code for it, and for this entire site here

Lets start simple, this is a static html polymer element that will show a blog card, with static content. Nothing much, just shows you the basic setup of a Polymer element. What we did here:

<dom-module id='waffi-element-1'>

<template>

...

...

</templat>

<script>

...

...

</script>

The dom-module part tells the browser that there is an element called waffi-element-1.

The template part defines the html contents of this element

The script part is used to specify properties and behaviour of this element.

Code should be easy to navigate, few things to note:

  • Webcomponent-lite.js is imported, otherwise this will only work on Chrome
  • An element will not see any CSS classes defined outside of itself, not will it affect anything outside, so every element must define/import its own styles
  • Check the function bindings done in attached function. This will make sure that using this in the event listeners will always give consistent results
  • The sample defines the element in the same html page that it is using it, this is not a good practice, see next part for a moe appropriate example
