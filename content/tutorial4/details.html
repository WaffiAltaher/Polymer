Here I created a different element, content-reader that will read the contents of this website from a backend. As I Am too lazy to create a REST api just for this tutorial, I just read the HTML files directly from the server, but everything will work in the same way when calling JSON, just specfy json instead of html in the handle-as section of the code.

The contet reader element will read a file: content.json from the backend, that file will tell us which URLs to call in order to get the content.It will the nread all the content and prepare it all in JSON objects that are passed to multiple waffi-element-4 elements.

This will show you how to invoke an API, how to get multiple polymer elements to interact via data binding, how to add elements based on backend response via dom-repeat, and it has some nice JS Promise tricks!
