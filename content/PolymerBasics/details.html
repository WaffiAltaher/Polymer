Sitepoint Tutorial

This one-page simple tutorial is quite old (Polymer 0.3.4), but the concepts there are explained nicely, use it for understanding the concepts rather than learning exact syntax

Also worth noting is that the tutorial explains the /deep/ and ::shadow selectors, which are now deprecated, also Vulcanize is explained in the tutorial, and is mentioned in the Polymer 1 documentation, but not Polymer 2

Google Web Fundamentals

Strictly speaking, this is not Polymer specific documentation, this is Google's way of explaining all the new fundamental concepts that Polymer is based upon

This might be too much to read in one go, bet is to keep it as a reference that you can refer to every time you come across a new concept

Polymer Official Docs

Official Polymer Docs, obviously a must read, however bear in mind that it is not really well documented, so this is not a one stop shop for all your needs, unfortunately.

You should also know that Polymer changed a lot between Polymer 1 and Polymer 2, so if you need to work on Polymer 1 then you should read both versions documentation
